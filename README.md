# 💰 Smart Expense Tracker

A modern and responsive **Expense Tracking Web Application** built using **HTML, CSS, and JavaScript**, designed to help users manage their income and expenses efficiently with real-time insights and visual analytics.

---

## 🚀 Features

* ✅ Add Income & Expenses
* ✅ Real-time Balance Calculation
* ✅ Category-based Tracking
* ✅ Search Transactions
* ✅ Delete Individual Transactions
* ✅ Clear All Transactions
* ✅ Monthly Overview Chart (Bar Graph)
* ✅ Data Persistence using LocalStorage
* ✅ Toast Notifications for Actions
* ✅ Fully Responsive UI

---

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript (Vanilla JS)
* **Charts:** Chart.js
* **Storage:** Browser LocalStorage

## ⚙️ Installation & Setup

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/expense-tracker.git
   ```

2. Navigate to the project folder:

   ```bash
   cd expense-tracker
   ```

3. Open `index.html` in your browser
   OR use **Live Server** in VS Code.

---

## 📊 How It Works

* Add transactions (income/expense)
* Data is stored in browser using LocalStorage
* Transactions are dynamically rendered
* Monthly data is visualized using Chart.js
* Summary cards update automatically

---

## 🧠 Key Concepts Used

* DOM Manipulation
* Event Handling
* Array Methods (`map`, `filter`, `reduce`)
* Local Storage API
* Dynamic UI Rendering
* Data Visualization
